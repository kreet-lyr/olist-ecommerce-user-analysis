# olist-ecommerce-user-analysis
# 跨境电商用户价值分层与精准营销数据分析
## 项目简介
基于巴西跨境电商平台Olist公开数据集，通过RFM用户分层模型完成用户价值划分，挖掘不同层级用户消费行为特征，输出差异化精准营销落地策略，解决电商盲目投放、营销成本浪费问题。

## 数据集来源
Kaggle公开数据集：https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce
数据集包含9张业务数据表，覆盖用户、订单、支付、物流、商品、评价全链路数据。

## 技术栈
Python｜Pandas｜NumPy｜Matplotlib/Seaborn｜Plotly｜Jupyter Notebook
数据分析：数据清洗、多表关联、RFM用户分层、用户画像、可视化分析、营销方案输出

## 项目核心流程
1. 数据清洗与多表融合
2. RFM指标构建与用户分层打分
3. 8类用户群体特征画像分析
4. 分层可视化消费行为对比
5. 分人群精准营销策略制定
6. 业务优化落地建议

## 项目成果
1. 完整用户分层结果表，区分平台核心高价值/流失/潜力用户
2. 多维度可视化图表，直观展示用户消费差异
3. 可直接落地的分人群精准营销方案，降低营销投放成本、提升复购转化
